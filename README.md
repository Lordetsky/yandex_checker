# CU Contest Checker 🚀

**CU Contest Checker** — это современный инструмент для преподавателей и ассистентов, позволяющий удобно отслеживать успеваемость студентов в Yandex Contest. Инструмент предоставляет наглядную аналитику, автоматический расчет оценок и удобный интерфейс для просмотра кода.

![UI Screenshot](https://img.icons8.com/clouds/100/000000/microscope.png)

## ✨ Основные возможности

- 🔍 **Умный многоуровневый поиск**: 
  - Автоматический маппинг студентов через `Students.xlsx` (поиск по фамилии $\to$ почта $\to$ Яндекс логин).
  - Модальное окно разрешения коллизий (если найдено несколько профилей).
  - Резервный поиск через транслитерацию и нечеткое сравнение (поиск с опечатками).
- 🔄 **Git-style Diff (Разница кода)**: Автоматический поиск предыдущей попытки студента и расчёт `unified diff` изменений (раскрашивается зеленым/красным).
- ⚡ **Ленивая загрузка (Lazy Loading)**: Список посылок загружается мгновенно, а код, логи тестов и разница версий подгружаются только при просмотре конкретного решения.
- 📊 **Интерактивная аналитика**: Графики распределения посылок по времени, статистика вердиктов и дедлайнов.
- 🎓 **Автоматический расчет оценки**: Если выбран конкретный студент, система автоматически считает процент решенных до дедлайна задач и выставляет балл (по 10-балльной шкале).
- 🌓 **Премиальный дизайн**: Современный интерфейс с удобной навигацией, Glassmorphism эффектами и подсветкой синтаксиса.
- 🛠 **Автоматизация запуска**: Скрипт `run.py` берет на себя всю рутину по настройке окружения и получению токенов.

## 🚀 Быстрый старт

Просто запустите одну команду в терминале (в папке проекта):

```bash
python run.py
```

Скрипт сам:
1. Создаст виртуальное окружение (`.venv`).
2. Установит все необходимые библиотеки.
3. Поможет получить OAuth-токен Яндекса (при первом запуске).
4. Запустит сервер и откроет доступ к приложению.

---

## 🛠 Ручная настройка (если нужно)

Если вы хотите настроить всё вручную:

1. **Создайте приложение в Яндекс ID**:
   - Перейдите на [oauth.yandex.ru](https://oauth.yandex.ru/).
   - Нажмите "Создать приложение".
   - Выберите платформу **Веб-сервисы**.
   - Callback URL: `https://oauth.yandex.ru/verification_code`.
   - В доступах выберите `contest:public_api` (или `contest:read`).

2. **Создайте файл `.env`.** в корне проекта:
   ```env
   YANDEX_TOKEN=ваш_полученный_токен
   ```

3. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Запустите сервер**:
   ```bash
   uvicorn main:app --reload
   ```

## 📖 Как пользоваться

1. **ID Контеста**: Введите ID контеста (его можно найти в URL контеста Яндекса).
2. **Дедлайн**: Укажите дату и время дедлайна. Веб-интерфейс автоматически пометит все поздние посылки.
3. **Поиск студента**:
   - Введите фамилию или логин в поле поиска.
   - Используйте подсказки автодополнения.
   - Нажмите «Поиск» или Enter.
4. **Просмотр решений**: Нажмите на строку с посылкой, чтобы раскрыть код и результаты тестов.
